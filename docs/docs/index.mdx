## Key Features

<div className="card-grid" style={{display: 'grid', gridTemplateColumns: 'repeat(auto-fit, minmax(300px, 1fr))', gap: '1rem', margin: '2rem 0'}}>

  :::tip Cluster Monitoring
Real-time health status, node statistics, and index metrics across all your
    clusters.
:::
  :::tip Index Management
Create, delete, and modify indices with visual feedback and detailed
    configurations.
:::
  :::tip Shard Management
Interactive grid-based shard allocation and relocation with visual
    representation.
:::
  :::tip REST Console
Execute Elasticsearch queries directly from the UI without leaving the
    application.
:::
  :::tip Multi-Cluster Support
Manage multiple Elasticsearch clusters from a single unified interface.
:::
  :::tip Multiple Auth Modes
Open mode, local users with bcrypt, and OIDC integration for enterprise
    deployments.
:::

</div>
